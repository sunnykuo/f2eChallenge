# FreeCell

## Functional map
* 新接龍的遊戲規則
* 額外功能：
	* 還原上一步
	* 計時遊玩時間

## Demo
* [Demo](https://sunnykuo.github.io/f2eChallenge/freeCell/#/)

## Step
* npm install
* npm run start
	* Runs the app in the development mode. Open http://localhost:9000 to view it in the browser.
* npm run build
	* Builds the app for production to the build folder.
